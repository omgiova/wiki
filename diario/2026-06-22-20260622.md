## Descobri: era Crawlee v1.7.2 (não Apify CLI) — 03:38

Quando disse "procura ai, instalei hj", o Hermes buscou no session_search e encontrou que **não foi hoje** — foi ontem (21/06) numa sessão longa de instalação de 10 repos. O pacote instalado relacionado ao Apify foi o **Crawlee v1.7.2** (`pip install crawlee`), framework open-source da Apify Technologies.

**O que veio junto:**
- `crawlee.crawlers` — 7 tipos de crawler (Http, BeautifulSoup, Parsel, Playwright, AdaptivePlaywright, Stagehand, Basic)
- `crawlee.proxy_configuration` — proxy rotativo
- `crawlee.fingerprint_suite` — anti-fingerprint/anti-bot
- `crawlee.sessions` — retry automático, sessões
- `crawlee._autoscaling` — escala automática conforme CPU/memória
- `crawlee.statistics` — métricas embutidas
- `crawlee._cli` — interface de linha de comando
- `crawlee.storages` / `crawlee.storage_clients` — filas, datasets, key-value stores

**Limitação:** Crawlee é o **motor** — não tem actors prontos pra Instagram. Pra Instagram rápido, precisa da plataforma cloud da Apify (API key + créditos). O framework serve pra construir scrapers customizados do zero.

**Insight de estilo:** usuário diz "instalei hoje" quando foi ontem — pode ser confusão de data/madrugada ou impressão subjetiva. O Hermes não deve corrigir, apenas buscar no histórico e mostrar o que encontrou.
